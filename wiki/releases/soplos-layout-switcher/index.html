Soplos Layout Switcher

Graphical utility for Soplos Boro (GNOME) that switches the desktop between predefined GNOME Shell layouts by enabling the required extensions and applying matching settings.

Versions

  • 1.0.0 — 2026-07-19

1.0.0 — July 2026

Initial release — four predefined GNOME Shell layouts, a Settings tab with quick access to GNOME tools, CLI support and full 8-language internationalization.

Added

  • Four layout presets: Soplos (Dash to Panel at the bottom with the Soplos orange indicator, ArcMenu in enterprise mode with Soplos branding, Desktop Icons NG and Quick Settings Tweaks — the default Boro look), Dock (Dash to Dock in floating bottom mode with Soplos theme applied), Tiling (Forge tiling engine, Dash to Dock on the left at 32 px without custom theme, Open Bar in Islands style with Soplos orange accent, AppIndicator, Blur my Shell and Workspace Indicator) and GNOME (stock GNOME Shell with all layout extensions disabled). All required extensions are managed automatically.
  • Settings tab: Quick access buttons for GNOME Tweaks, GNOME Extensions manager, GDM Settings and Firefox theming (Add Water), plus toggles for the system tray and desktop icons extensions.
  • CLI entry point (soplos-layout-switcher-cli): Apply any layout without opening the GUI — useful for installers and scripts (apply-soplos, apply-dock, apply-gnome, apply-tiling, reload-gnome-shell).
  • Full internationalization: Complete 8-language support (ES, EN, FR, DE, PT, IT, RO, RU) via gettext.
  • Soplos application skeleton: Header bar, footer status bar and About dialog consistent with the rest of the Soplos app ecosystem.
