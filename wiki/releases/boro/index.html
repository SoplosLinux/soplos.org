Hardware support

Over 2600 hardware drivers are embedded in the initrd for maximum out-of-the-box compatibility on bare-metal systems:

  • Wi-Fi: MediaTek (mt76 family), Intel (iwlwifi), Realtek (rtl8xxxu, rtl819x family), Broadcom (brcmfmac, b43), Atheros (ath9k, ath10k, ath11k, ath12k)
  • GPU: Intel (i915, Xe), AMD (amdgpu, radeon), NVIDIA (nouveau open-source; proprietary driver added in Alpha 1.0.1)
  • Storage: NVMe, SATA (ahci), USB mass storage, UAS
  • Bluetooth: MediaTek, Broadcom, Atheros
  • Ethernet: Full set of kernel Ethernet controllers
  • Firmware: MediaTek, Intel, AMD (amdgpu/radeon), Realtek, Atheros, Broadcom firmware bundles included
  • NVIDIA patches: Custom Soplos patches (nvidia-patches) for NVIDIA 550/580 DKMS compatibility with Linux kernel 6.19+ and 7.0

Aesthetics

  • GTK theme: Orchis-Orange-Dark-Compact
  • Icon theme: Tela-ubuntu-dark
  • Wallpapers: 24 Soplos backgrounds (/usr/share/backgrounds/soplos/)
  • Default wallpaper: soplos/006.jpg
  • GDM3: Custom login screen — Soplos background with blur effect and Soplos brand colours

GNOME Shell Extensions

20 extensions installed. Active by default:

  • ArcMenu — application launcher menu
  • Dash to Panel — taskbar replacing the GNOME dash
  • Clipboard Indicator — clipboard history manager
  • CoverflowAltTab — Alt+Tab switcher with cover-flow effect
  • Desktop Cube — 3D workspace cube animation
  • DING — Desktop Icons NG (file icons on desktop)
  • Quick Settings Tweaks — enhanced quick-settings panel

Also installed (not active by default): Apps Menu, Auto Move Windows, Drive Menu, Launch New Instance, Light Style, Native Window Placement, Places Menu, Screenshot Window Sizer, System Monitor, User Theme, Window List, Windows Navigator, Workspace Indicator.

Flatpak

  • Flatpak runtime installed (1.16.1). No system Flatpaks pre-installed.
