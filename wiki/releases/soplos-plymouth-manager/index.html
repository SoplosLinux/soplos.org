2.0.0 — March 2026

Major release unifying Tyron and Tyson branches — complete modular rewrite, Wayland support, advanced Xephyr previews, smart dark theme detection, robust theme installer and full 8-language internationalisation.

Added

  • Complete modular architecture: Codebase restructured into core/, ui/, utils/, config/, locale/ and assets/.
  • Unified environment detection (core/environment.py): Accurate detection of Desktop Environment, display protocol and system theme before privilege elevation.
  • Advanced KDE dark theme detection: Background mathematical brightness analysis and LookAndFeelPackage inspection for reliable Plasma dark mode detection.
  • Advanced GNOME & KDE previews: Xephyr bridge for Wayland sessions; xterm fallback for TTY-based themes.
  • Single-password prompt: Unified pkexec logic minimises privilege prompts across all Wayland sessions.
  • Responsive gallery: Dynamically adapts column count to window width with no rigid limits.
  • Standard keyboard shortcuts: Ctrl+Q, F5, Ctrl+A, Ctrl+P, Ctrl+I, Delete.
  • Robust theme installer: Deep recursive search for compressed archives with nested folder structures (.tar.gz, .tar.xz, .tar.bz2, .zip).
  • Full dynamic theme detection: Eliminates all hardcoded theme names and module lists.
  • Complete internationalisation: Native support for 8 languages (ES, EN, DE, FR, IT, PT, RO, RU) via gettext.

Changed

  • App ID: Changed to org.soplos.plymouthmanager.
  • Total visual renewal following Soplos v2.0 design standards.
  • Progress bar: Relocated to a dynamic Revealer above the footer with smooth updates during initramfs generation.
  • Dynamic protection of active and critical system themes to prevent accidental removal.

Improved

  • Native Dracut support: Correct initramfs regeneration on Soplos Linux (Dracut-based).
  • Standard-compliant logging: Log files stored at ~/.cache/soplos-plymouth-manager/.
  • Thumbnail centering and scaling: Premium alignment of theme previews in the gallery.

Fixed

  • Duplicated footer removed.
  • Asymmetric preview centering corrected.
  • CSS syntax errors for GTK 3 compatibility fixed.
  • Progress bar stuck at 20% during long initramfs operations fixed.
  • Installation issues with nested folder structures in theme bundles fixed.
