Soplos WebApp Manager

Site Specific Browser engine for Soplos Linux — turn any website into a standalone desktop app with isolated browser profiles, multi-engine support and full internationalization.

Versions

  • 1.0.0-5 — 2026-07-07
  • 1.0.0-4 — 2026-04-30
  • 1.0.0-3 — 2026-03-22
  • 1.0.0-2 — 2026-03-21
  • 1.0.0-1 — 2026-03-10
  • 1.0.0 — 2026-03-04

1.0.0-5 — July 2026

Build dependency update — python3-all replaced with python3 in debian/control.

Changed

  • Build dependency python3-all replaced with python3.

1.0.0-4 — April 2026

Footer text simplified to Soplos standard, icon persistence improved and system theme support fixed.

Changed

  • Footer text: Simplified to match the modern Soplos standard layout (XFCE · X11).

Fixed

  • Icon Persistence: WebApp icons are now saved permanently in the local data folder instead of the cache folder, preventing icon loss during system cleanups.
  • Theme Support: Fixed dark mode being forced on light themes; the application now correctly respects the system's light or dark GTK theme preference.

1.0.0-3 — March 2026

Background color inconsistency fixed to ensure a correct and uniform Soplos dark theme appearance.

Fixed

  • UI colors: Fixed background color inconsistency — window, list and scrolled area now consistently use the correct Soplos dark theme color (#2b2b2b).

1.0.0-2 — March 2026

About dialog added, accessible via F1 or the GNOME application menu.

Added

  • About dialog: Press F1 or use the GNOME application menu to open the About dialog with version, author, license and website.

1.0.0-1 — March 2026

Standardized Soplos UI with official footer, environment detection, Incognito mode and extended browser parameter help.

Added

  • Official Soplos Footer: Implemented a standard status bar following the Soplos ecosystem standard, with automatic DE and protocol detection.
  • Environment Detection: Added automatic identification of Desktop Environment (GNOME, Plasma, XFCE) and Protocol (Wayland, X11).
  • Incognito Mode: New support for private browsing sessions with fully isolated profiles.
  • Extended Parameters Help: Added a comprehensive guide for advanced browser flags and parameters.
  • UI Alignment: Consistent use of dim-label styling and standard margins (15, 15, 8, 8) for seamless transition between Soplos apps.

1.0.0 — March 2026

Initial release — full Site Specific Browser engine with multi-engine support, WebApp editing, Wayland compatibility, isolated Firefox profiles, smart favicons and 8-language internationalization.

Added

  • Multi-Engine Support: Native or Flatpak versions of Firefox, Chrome, Chromium, Brave, Vivaldi, Edge and Epiphany.
  • Full WebApp Editing: Users can edit Name, URL, Icon, Browser and Navigation Bar settings for existing WebApps at any time.
  • Extra Browser Parameters: Support for custom browser flags (e.g. --start-maximized, --kiosk) at both creation and editing.
  • Improved Wayland Compatibility: Enhanced icon mapping and window association for KDE Plasma 6 and other modern Wayland environments.
  • Independent Firefox Instances: Optimized Firefox SSB behavior with truly isolated processes, dedicated icons and userChrome.css customization.
  • Smart Favicons: Fallback mechanism to download high-quality website icons via Google Favicons API.
  • Full Internationalization (i18n): Translated for the 8 default Soplos Linux languages (ES, EN, FR, DE, PT, IT, RO, RU).
