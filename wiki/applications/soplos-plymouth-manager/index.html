Soplos Plymouth Manager

Customize Soplos Linux's boot animation from a visual gallery with real-time preview support.

Contents

  • Overview
  • Features
  • Screenshots

Overview

Soplos Plymouth Manager is a robust GTK3 graphical application that serves as a central hub for managing your boot (Plymouth) themes. Beyond a simple selector, the tool dynamically analyzes your system's configuration to generate visual thumbnails for each design, bringing them together in a responsive, interactive gallery.

You can effortlessly import packaged themes straight from the interface, preview exactly how the boot animation will look on your screen, and apply your choice with an initramfs rebuild running in the background. The tool also includes intelligent safeguards to prevent accidental removal of active or critical system animations.

Key Features

  • Smart Visual Gallery: Browse all your available themes through dynamic cards generated from real-time module data and thumbnails.
  • Isolated Preview Engine: Want to see how your splash screen looks before rebooting? The app launches a temporary, isolated display using Xephyr to emulate the true loading animation, supporting both X11 and Wayland display protocols.
  • Streamlined Installer: Easily import compressed tarball graphics. The app recursively detects Plymouth configuration files and moves them exactly where they need to be in the system tree.
  • Initramfs Application Process: Setting an active theme triggers deep system integration, with real-time progress tracking as it updates and reconstructs the initial ramdisk (initramfs) modules.
  • Context-Aware Environment: Built on the Soplos 2.0 standard, it seamlessly integrates its window decorations whether you use XFCE's native window manager or the modernized GNOME and Plasma frameworks.
