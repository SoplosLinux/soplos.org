Exclusive Applications for Boro

The Boro (GNOME) edition includes a set of tools developed by the Soplos Linux team to facilitate system configuration and customization.

Soplos AppImage Manager

Portable application manager for AppImage files. Integrate, launch and update AppImage packages with desktop and menu support.

  • Automatic desktop integration with icon extraction
  • Built-in update checker for installed AppImages
  • Property editor with custom icon and name support
Learn More

Soplos GRUB Editor

Graphical interface to customize the GRUB bootloader. Safely manage boot entries, themes, and system settings with advanced backend integration.

  • Advanced environment detection for dynamic Wayland/X11 theming
  • Hierarchical boot entry management with nested submenus
  • Robust theme installation and intelligent GRUB cache cleanup
Learn More

Soplos Kernel Installer

Build and install custom Linux kernels with a graphical interface. Select kernel profiles and patches optimized for gaming, audio, or minimal setups.

  • Multiple profiles: Gaming, Audio/Video, Minimal/Office, Soplos Stock
  • Patch support: BORE, PREEMPT_RT, Zen, NTSYNC
  • One-click Soplos Kernels repository integration
Learn More

Soplos Layout Switcher

Switch between four predefined GNOME Shell layouts with automatic extension management.

  • Four layouts: Soplos, Dock, Tiling and GNOME
  • Automatic extension enable/disable per layout
  • Settings tab with quick access to GNOME tools
Learn More

Soplos Plymouth Manager

Give your machine's boot-up a unique visual touch using an intuitive interface equipped with live previews. Install, apply, and remove themes without touching the terminal, while the app manages all internal rebuild operations.

  • Visual gallery with recursive theme detection
  • Isolated preview engine for GNOME environments
  • Integrated initramfs update progress tracking
Learn More

Soplos Repo Selector

Manage APT repositories and DEB822 sources easily. Includes a built-in speed test to automatically select the fastest mirrors.

  • Built-in mirror speed test
  • Automatic conversion of legacy lists to DEB822
  • Atomic PKEXEC operations and GPG management
Learn More

Soplos System Service

GTK3 systemd service manager — view, control and monitor services with color-coded status, Start/Stop/Restart controls and journalctl log viewer.

  • Color-coded service list (green active, red failed)
  • Start, Stop and Restart with smart state detection
  • Journalctl log viewer with monospace display
Learn More

Soplos Sys Cleaner

System cleaner for removing orphaned packages, cache, logs and temporary files. Keeps your system fast and lean.

  • Package cache and orphaned dependency cleanup
  • Log and temporary file management
  • Disk usage analysis
Learn More

Soplos WebApp Manager

Create and edit complete standalone desktop applications from URLs with independent profiles and multiple browser engines.

  • Interactive Multi-Engine support with full editing
  • Isolated browser profiles and Incognito Mode
  • Advanced boot parameters and smart HD Favicons
Learn More

Soplos Welcome

Post-installation assistant with modular architecture adapted to GNOME. Manages drivers, kernels, software, security and desktop customization.

  • Complete hardware detection and driver installation
  • Advanced package management and security tools
  • Native GNOME customization integration
Learn More

Soplos Welcome Live

Live session companion for installing Soplos Linux or recovering existing systems via CHROOT with BTRFS support.

  • Direct Calamares installer integration
  • Advanced CHROOT with partition and subvolume detection
  • Language switching with XDG folder migration
Learn More
