Soplos Welcome

Welcome application for Soplos Linux. Initial system setup, recommended software installation and desktop customization.

Features

Initial Setup

Guides the user through initial system configuration after first boot.

Recommended Software

Install recommended applications by category (Office, Developer, Multimedia, etc.).

Desktop Customization

Apply themes, set wallpapers and configure desktop preferences.

Hardware Detection

Detects hardware and suggests drivers and firmware when needed.

Installation

Soplos Welcome is included by default in Soplos Tyron. To reinstall or update:

Terminal

sudo apt update
sudo apt install soplos-welcome

Usage

  • Open the application: Run soplos-welcome from the menu.
  • Follow first-boot steps: Configure timezone, keyboard, and user preferences.
  • Install recommended software: Use the Recommended tab to install or remove curated packages.
  • Customize: Apply themes, wallpapers and shortcuts using the customization tab.
  • Advanced: Use the hardware detector for driver recommendations and the logs for debugging.

    FAQ & Troubleshooting

    How do I use the Recommended tab?

    Use the Recommended tab to install or remove curated packages by category. The install buttons will update their state after operations.

    How does hardware detection work?

    The hardware detector scans system devices and recommends drivers; it includes heuristics for NVIDIA and common hardware families.
