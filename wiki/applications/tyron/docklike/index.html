Soplos Docklike

Soplos Docklike is a graphical manager for the Docklike plugin (XFCE). It helps you add, remove, reorder and configure the application launchers (desktop entries) used by the Docklike plugin.

Features

Icon Management

Soplos Docklike provides a graphical manager to add, remove and reorder icons used by the Docklike plugin in XFCE.

Configure Entries

Edit application entries (paths and desktop files) and customize which launchers appear in the dock.

Multi-language

Includes translations and uses system locale to display UI text in multiple languages (ES, EN, FR, PT, DE, IT, RU, RO).

Save & Apply

Save changes to Docklike configuration and optionally restart the XFCE panel to apply updates.

Installation

Soplos Docklike is included in the default installation of Soplos Tyron. If you need to reinstall or update it:

Terminal

sudo apt update
sudo apt install soplos-docklike

After installation, you can add it to your XFCE panel through the Panel preferences.

Usage

  • Open the manager: Run `soplos-docklike` from the application menu or via the command `soplos-docklike` (installed via package).
  • Select applications: Use the selector to pick applications from system `.desktop` files; icons are loaded from system icon themes or provided paths.
  • Reorder and edit: Move icons up/down, remove entries, or edit properties in the list view.
  • Save and apply: Click "Save" to write the configuration. The tool can restart the XFCE panel to apply changes if needed.
  • Advanced: Configuration is written to the Docklike rc files under `~/.config/xfce4/panel` and desktop entries are read from standard application directories.

    FAQ & Troubleshooting

    What is Soplos Docklike?

    Soplos Docklike is a graphical manager for the Docklike XFCE plugin. It lets you add, remove, reorder and edit the application launchers (.desktop) shown in the dock.

    Where is the configuration stored?

    Configuration is written to the XFCE panel configuration files, typically under ~/.config/xfce4/panel (files named like docklike-*.rc).

    How do I apply changes?

    After saving, the tool can offer to restart the XFCE panel so changes take effect. Alternatively, restart the panel or your session manually.

    What if icons are missing or broken?

    Docklike loads icons from the system icon theme or from absolute paths specified in .desktop files. Make sure the icon files exist in standard directories (for example /usr/share/icons or ~/.local/share/icons). Use the editor to select a valid icon if necessary.
