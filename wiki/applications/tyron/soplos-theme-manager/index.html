Soplos Theme Manager

A simple, intuitive theme manager for Soplos Tyron (XFCE): create, save and apply desktop themes; install icons and manage backups.

Features

Theme Creation

Create, save and apply complete themes (panel, xfwm, wallpapers, shortcuts).

Icon & Theme Packages

Install and manage icon sets and theme packages with a simple UI.

XFCE Integration

Optimized compatibility with XFCE/Tyron and writes config files where needed.

Backups & Restore

Automatic theme backups and easy restore from the UI (.themes-backup/ included).

Previews

Theme previews with thumbnails and full previews (preview.png / preview_full.png).

Multilanguage

Full translations: ES, EN, FR, PT, DE, IT, RU, RO.

Installation

Install Soplos Theme Manager from Soplos repositories or manually:

Terminal

sudo apt update
sudo apt install soplos-theme-manager

Also available as a Python package in the project's repository (see setup.py for dependencies).

Usage

  • Open the app: Run soplos-theme-manager from the menu or terminal.
  • Browse themes: Use the UI to preview themes and thumbnails.
  • Create / Save: Customize panel, wallpaper and style; save the theme to backups.
  • Apply: Select a theme and click apply; the app handles backups and writes the necessary XFCE configs.
  • Debugging: Includes debugging helpers (for example debug_wmclass.py) and logs to ~/.config/soplos-theme-manager/logs.

    FAQ & Troubleshooting

    Where are the logs stored?

    Logs are stored at ~/.config/soplos-theme-manager/logs/theme-manager.log.

    Can the app be run as root?

    No — the application prevents running as root for safety. Run as a normal user; the app will prompt for elevation when needed.
