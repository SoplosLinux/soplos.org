Soplos Welcome Live

Live session variant of the Welcome assistant: chroot recovery, partition detection and first-boot tools optimized for live media.

Features

Advanced Partition Detection

Detect and select partitions and btrfs subvolumes for chroot operations.

CHROOT Recovery

Tools to perform chroot recovery operations safely from live media.

Internationalization

Full language support with dictionaries split per language for easy maintenance.

Robust Recovery Flow

Improved compatibility with multiple terminals and recovery environments.

Installation

Soplos Welcome Live is typically included in the live image. For packaging or updates, install the package:

Terminal

sudo apt update
sudo apt install soplos-welcome-live

Usage

  • Boot live media: Start the Soplos Tyron live session and run soplos-welcome-live.
  • Select partition: Use the partition selector to choose the correct root or subvolume for chroot.
  • Perform recovery: Use CHROOT tools to repair or recover an installed system.
  • Internationalized UI: Choose your language; dictionaries are loaded per language.

    FAQ & Troubleshooting

    How do I detect partitions correctly?

    Use the partition selector carefully and verify the detected devices before applying CHROOT operations.

    Is btrfs supported?

    Yes — Welcome Live supports btrfs subvolumes and provides options to mount them correctly in CHROOT.
