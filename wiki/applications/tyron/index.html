Exclusive Applications for Tyron

The Tyron (XFCE) edition includes a set of tools developed by the Soplos Linux team to facilitate system configuration and customization.

Soplos Docklike

A modern dock for XFCE inspired by macOS. Features include auto-hide, icon magnification, and customizable themes.

  • Auto-hide functionality
  • Icon zoom on hover
  • Customizable position and size
Learn More

Soplos GRUB Editor

Graphical interface to customize the GRUB bootloader. Safely manage boot entries, themes, and system settings with advanced backend integration.

  • Advanced environment detection for dynamic Wayland/X11 theming
  • Hierarchical boot entry management with nested submenus
  • Robust theme installation and intelligent GRUB cache cleanup
Learn more

Soplos Plymouth Manager

Boot splash screen manager for customizing Plymouth themes. Install, preview, and apply beautiful boot animations with an intuitive graphical interface.

  • Dynamic visual gallery with generated thumbnails
  • Real-time preview engine using isolated Xephyr screens
  • Automatic initramfs rebuilding and path updating
Learn more

Soplos Repo Selector

Manage APT repositories and DEB822 sources easily. Includes a built-in speed test to automatically select the fastest mirrors.

  • Built-in mirror speed test
  • Automatic conversion of legacy lists to DEB822
  • Atomic PKEXEC operations and GPG management
Learn more

Soplos Theme Manager

Apply and manage GTK themes, icons, and cursors for XFCE. Browse and install themes from online sources.

  • Install GTK themes
  • Change icon sets
  • Customize cursor themes
Learn more

Soplos Welcome

Post-installation assistant with modular architecture adapted to XFCE. Manages drivers, kernels, software, security and desktop customization.

  • Complete hardware detection and driver installation
  • Advanced package management and security tools
  • Native XFCE customization integration
Learn more

Soplos Welcome Live

Live session companion for installing Soplos Linux or recovering existing systems via CHROOT with BTRFS support.

  • Direct Calamares installer integration
  • Advanced CHROOT with partition and subvolume detection
  • Language switching with XDG folder migration
Learn more

Soplos WebApp Manager

Transform websites into independent desktop applications with isolated profiles and menu integration.

  • Creates standalone app windows
  • Isolated browsing profiles
  • Custom icon support
Learn more
