Exclusive Applications for Tyron

The Tyron (XFCE) edition includes a set of tools developed by the Soplos Linux team to facilitate system configuration and customization.

Soplos Docklike

A modern dock for XFCE inspired by macOS. Features include auto-hide, icon magnification, and customizable themes.

  • Auto-hide functionality
  • Icon zoom on hover
  • Customizable position and size
Learn More

Soplos GRUB Editor

Graphical interface to customize the GRUB bootloader. Easily change boot order, themes, and timeout settings.

  • Change default boot entry
  • Apply custom GRUB themes
  • Set boot timeout
Learn more

Soplos Plymouth Manager

Manage and customize Plymouth boot splash screens. Preview and apply themes with a simple interface.

  • Preview Plymouth themes
  • Apply custom splash screens
  • Backup and restore settings
Learn more

Soplos Repo Selector

Select and configure software repositories intuitively. Add official and third-party repos with ease.

  • Add/remove repositories
  • Enable/disable sources
  • Update package lists
Learn more

Soplos Theme Manager

Apply and manage GTK themes, icons, and cursors for XFCE. Browse and install themes from online sources.

  • Install GTK themes
  • Change icon sets
  • Customize cursor themes
Learn more

Soplos Welcome

Welcome assistant that guides you through first steps with Tyron. Includes tutorials and system info.

  • System overview
  • Basic configuration tips
  • Links to documentation
Learn more

Soplos Welcome Live

Special version of the welcome assistant for Tyron live sessions. Optimized for demo and testing.

  • Live session info
  • Installation prompts
  • System requirements check
Learn more
