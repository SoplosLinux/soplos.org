Soplos GRUB Editor

Graphical editor for GRUB2: manage boot entries, themes, timeout and create safe backups before applying changes.

Features

Edit Boot Entries

Add, remove or reorder GRUB boot entries and set the default entry.

Theme Management

Install and preview GRUB themes, set background images and fonts.

Timeout & Defaults

Configure GRUB timeout, default entry, and kernel parameters easily.

Backups & Restore

Automatic backups before applying changes and easy restore options.

Installation

Soplos GRUB Editor is available in Soplos repositories. To install:

Terminal

sudo apt update
sudo apt install soplos-grub-editor

Usage

  • Open the application: Run soplos-grub-editor from the menu.
  • Edit entries: Add, rename or reorder boot entries and save changes.
  • Apply and backup: The app creates a backup before writing changes to GRUB.
  • Themes & fonts: Preview and apply GRUB themes and fonts from the UI.

    FAQ & Troubleshooting

    What does Soplos GRUB Editor do?

    It provides a GTK graphical interface to edit GRUB2 configuration, themes, timeouts and manage boot entries safely with automatic backups.

    Is it safe to apply changes?

    The app creates backups before applying changes and validates configurations. If something goes wrong, you can restore the backup.
