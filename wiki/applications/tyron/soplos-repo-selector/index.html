Soplos Repo Selector

Graphical tool to select, add and manage APT repositories and sources lists on Soplos Tyron (XFCE).

Features

Add / Remove Repos

Add or remove repository .list files and manage sources easily.

Enable / Disable

Enable or disable individual repository entries without editing files manually.

Import Keys

Import third-party repository GPG keys by URL or local file.

Update & Backup

Run apt update from the UI and create/restore backups of sources.list.

Installation

Install Soplos Repo Selector from Soplos repositories or manually:

Terminal

sudo apt update
sudo apt install soplos-repo-selector

Usage

  • Open the app: Run soplos-repo-selector from the menu.
  • Add a repo: Click "Add" and provide the APT line (for example: deb http://deb.example.com stable main).
  • Import key: When required, import the repository GPG key via the UI or paste the key file.
  • Update: Use the "Refresh" button to run apt update and show results in the UI.
  • Backup: Create or restore backups of your sources before applying changes.

    FAQ & Troubleshooting

    How do I add a repository that requires a GPG key?

    Use the "Import key" option in the dialog. The app supports fetching keys by URL or importing a local key file.

    Can I revert changes if something breaks?

    Yes — the app creates a backup of your sources before applying changes and provides a restore option.
