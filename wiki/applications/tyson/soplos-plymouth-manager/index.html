Soplos Plymouth Manager

Manage and customize Plymouth boot splash screens for Tyson (KDE Plasma).

Features

Preview Themes

Preview Plymouth themes before applying them to the system.

Backup & Restore

Backup current Plymouth configuration and restore if needed.

Integration

Integrates with system settings and supports Wayland/X11 environments.

Installation

Install from the official repositories or via the Debian package:

Terminal

sudo apt update
sudo apt install soplos-plymouth-manager

Usage

  • Open the app: Run soplos-plymouth-manager from the menu.
  • Preview: Select a theme and preview before applying.
  • Backup: Create a backup before making changes.
