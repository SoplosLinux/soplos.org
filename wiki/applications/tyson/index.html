Exclusive Applications for Tyson

The Tyson (KDE Plasma) edition includes a set of tools developed by the Soplos Linux team to facilitate system configuration and customization.

Soplos GRUB Editor

Graphical interface to customize the GRUB bootloader. Easily change boot order, themes, and timeout settings.

  • Change default boot entry
  • Apply custom GRUB themes
  • Set boot timeout
Learn more

Soplos Plymouth Manager

Manage and customize Plymouth boot splash screens. Preview and apply themes with a simple interface.

  • Preview Plymouth themes
  • Apply custom splash screens
  • Backup and restore settings
Learn more

Soplos Repo Selector

Select and configure software repositories intuitively. Add official and third-party repos with ease.

  • Add/remove repositories
  • Enable/disable sources
  • Update package lists
Learn more

Soplos Welcome

Welcome assistant that guides you through first steps with Tyson. Includes tutorials and system info.

  • System overview
  • Basic configuration tips
  • Links to documentation
Learn more

Soplos Welcome Live

Special version of the welcome assistant for Tyson live sessions. Optimized for demo and testing.

  • Live session info
  • Installation prompts
  • System requirements check
Learn more
