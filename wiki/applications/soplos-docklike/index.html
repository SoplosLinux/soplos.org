Soplos Docklike

A modern, icon-only taskbar plugin for XFCE, bringing a dock-like experience.

Contents

  • Overview
  • Features
  • Screenshots

Overview

Soplos Docklike is a specialized graphical tool designed to manage and organize pinned applications in the XFCE panel's Docklike plugin. It provides a user-friendly interface to customize your dock without manually editing configuration files, making it easy to keep your most-used apps at your fingertips.

With Soplos Docklike, you can effortlessly add new applications, remove unused ones, and reorder your dock items to suit your workflow. It integrates seamlessly with the XFCE desktop environment, ensuring a native and responsive experience.

Key Features

  • Visual Management: Add, remove, and reorder pinned applications with a clean graphical interface.
  • Drag & Drop Support: Intuitively reorder your icons using drag and drop functionality.
  • Application Selector: Built-in dialog to easily find and add any installed application to your dock.
  • Instant Application: Automatically restarts the XFCE panel to apply changes immediately without a session restart.
  • Keyboard Shortcuts: Full keyboard control for power users (Ctrl+S to save, Alt+Arrows to move items, Delete to remove).
  • Multi-language Support: Fully translated into 8 languages (English, Spanish, French, Portuguese, German, Italian, Russian, Romanian).
