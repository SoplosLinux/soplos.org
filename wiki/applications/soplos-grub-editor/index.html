Soplos GRUB Editor

Customize your boot experience. Change the default OS, timeout, and visual themes safely.

  • Overview
  • Features
  • Screenshots

Overview

Soplos GRUB Editor is an advanced graphical tool for configuring the GRUB2 bootloader. It allows you to customize boot entries, themes, and kernel parameters without editing complex configuration files.

Compatible with XFCE, Plasma, and GNOME, it provides a safe and easy way to manage your boot options, including automatic backups to prevent configuration errors.

Key Features

  • Visual Configuration: Edit GRUB settings through a user-friendly GTK3 interface.
  • Theme Management: Install and apply GRUB themes, change fonts, and set custom background images.
  • Boot Entry Management: Add, edit, or remove custom boot entries with ease.
  • Backup & Restore: Automatically backs up configuration before applying changes for peace of mind.
  • Wayland Support: Fully compatible with Wayland on Plasma, GNOME, and other compositors.
  • Kernel Parameters: Advanced configuration for kernel boot parameters.
