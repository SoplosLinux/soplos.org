Soplos Welcome Live

The essential guide for your live session. Install Soplos Linux or explore the system.

Contents

  • Overview
  • Features
  • Screenshots

Overview

Soplos Welcome Live is the specialized welcome screen for the Soplos Linux Live environment. It helps users navigate the live system, perform the installation, or recover an existing system.

Beyond just a welcome screen, it includes powerful tools for system administrators and advanced users, such as a robust CHROOT recovery environment and intelligent partition detection.

Key Features

  • Direct Installation: Seamless integration with the Calamares installer for a smooth setup.
  • System Recovery: Advanced CHROOT tools to recover broken systems, with intelligent partition detection.
  • Language Configuration: Automatically configures the system language and keyboard layout based on your selection.
  • XDG Folder Migration: Smart migration of user folders (Documents, Downloads, etc.) when changing languages.
  • NumLock Management: Control NumLock state for the live session and the installed system.
  • GParted Integration: Quick access to partition management tools for disk preparation.
