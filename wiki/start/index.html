System Updates

Keeping your system updated is crucial for security, stability, and getting the latest features.

Via Soplos Welcome

  1. Open Soplos Welcome from the applications menu
  2. Click on "Check for Updates"
  3. Review available updates with descriptions
  4. Click "Install Updates" and enter your password

Via Terminal (Advanced Users)

Update Commands

# Update package list
sudo apt update

# Upgrade all packages
sudo apt upgrade

# Full system upgrade
sudo apt full-upgrade
Important: Always reboot after kernel updates to ensure all changes take effect.

Software Centers

Software Tab: Soplos Welcome includes a dedicated Software management section for installing package managers and software centers.

The Software tab in Soplos Welcome allows you to install and manage various software centers and package managers:

Available Software Centers

  • Synaptic Package Manager: Advanced APT package manager with GUI
  • Gdebi: Simple tool to install local .deb packages
  • GNOME Software: User-friendly software store (Default in Boro) with Flatpak and Snap support
  • Flatpak Support: Universal package system with Plasma Discover integration
  • Snap Support: Ubuntu's universal packages with Plasma Discover support

System Maintenance

  • Update Repositories: Refresh package lists from all configured sources
  • Upgrade System: Install all available package updates
  • Clean System: Remove unnecessary packages and cached files

Repository Management

  • Soplos Repo Selector: Graphical tool to manage and optimize APT repositories
  • Add Flathub: Enable the main Flatpak repository
  • Snap Menu Integration: Integrate Snap applications into the system menu
One-Click Installation: All software centers can be installed with a single click, including their dependencies and integration packages.

How to Install Applications

  1. Open Soplos Welcome from the applications menu
  2. Open Soplos Welcome from the applications menu
  3. Navigate to the "Recommended" tab
  4. Browse the application categories
  5. Click "Install" on any application you want
  6. The app will handle dependencies and package management automatically

Hardware Drivers

Drivers Tab: Soplos Welcome automatically detects your hardware and installs the necessary proprietary drivers for optimal performance.

NVIDIA Drivers

  • Latest Driver (Series 500+): For GeForce GTX 1600+, RTX 2000+, RTX 3000+, RTX 4000+
  • Legacy 470: For GeForce GTX 900, GTX 1000
  • Legacy 390: For GeForce GTX 600, GTX 700, GTX 800
  • Legacy 340: For GeForce GTX 400, GTX 500
  • Nouveau (Open Source): Free and open source NVIDIA driver
  • DaVinci Resolve Support: OpenCL and video encoding support
  • Blender/CUDA Support: CUDA toolkit for rendering

AMD Drivers

  • AMD Radeon (AMDGPU): Modern open source driver for RX 400+ series and APUs
  • Radeon (Legacy): Legacy open source driver for older cards
  • Mesa & Vulkan: Complete graphics stack with firmware support

Intel Graphics

  • Intel i915: For Intel HD, UHD, Iris, and Iris Xe graphics
  • Intel Xe: For Intel Arc discrete graphics cards

Wi-Fi Drivers

  • MediaTek: MT76xx series (mt7601u, mt76, mt7915e, mt7921u, mt7925e, etc.)
  • Intel WiFi: iwlwifi, iwldvm, iwlmvm for AC/AX series and legacy chips
  • Realtek: rtl8xxxu, rtl8192xx, rtl8188ee, rtl8723xx, rtl8821ae series
  • Broadcom: brcmfmac, brcmsmac, b43, b43legacy for BCM43xx chips
  • Atheros: ath9k, ath10k, ath11k, ath12k, carl9170 for all Atheros series

Ethernet Drivers

  • Complete Support: All available Ethernet drivers from kernel modules
  • Realtek, Intel, Broadcom: Full coverage for desktop and laptop NICs

Bluetooth Support

  • Universal Support: btusb, btmtk, btbcm for MediaTek and Broadcom chips
  • Atheros Support: ath3k for Atheros Bluetooth devices
  • UART Support: hci_uart for embedded Bluetooth modules

Printer Support

  • Universal Drivers: Support for HP, Canon, Epson, Brother and major brands

Storage Support

  • NVMe: Complete NVMe SSD support with nvme-core drivers
  • SATA/AHCI: Full SATA and AHCI controller support
  • USB Storage: Universal USB mass storage and UAS support

Virtual Machine Tools

  • VMware Tools: Open VM Tools Desktop for VMware Workstation/Player integration
  • QEMU/KVM Agent: Guest agent and tools for QEMU/KVM virtual machines
  • VirtualBox Guest Additions: Complete VirtualBox integration with shared folders and clipboard
Maximum Hardware Compatibility: Soplos Linux includes an extensive driver collection for maximum hardware compatibility across desktops, laptops, and virtual machines.

Kernel Management

Kernels Tab: Install, remove or clean alternative kernels like Liquorix and XanMod to improve system performance.

Available Kernels

Liquorix Kernel

Kernel optimized for desktop and gaming, with low latency and performance improvements.

XanMod Kernel

Kernel optimized for low latency and high performance, ideal for demanding tasks.

System Maintenance

  • Clean Old Kernels: Remove old kernels and free up disk space
  • Update GRUB: Update GRUB configuration after kernel changes
  • Current Kernel Info: Display details about the kernel in use
NVIDIA Compatibility: Alternative kernels may require NVIDIA driver reinstallation after updates.

Desktop Customization

  • Settings → Appearance

  • Configure Dash to Panel settings

Multimedia Support

Pre-configured: Soplos Linux comes with all multimedia codecs already installed and configured. You can play most media formats out of the box without additional setup.

What's Already Included

  • Audio Codecs: MP3, AAC, FLAC, OGG, Opus, and more
  • Video Codecs: H.264, H.265, VP9, AV1, and proprietary formats
  • Container Formats: MP4, MKV, AVI, WebM, MOV, and others
  • Streaming Support: Ready for Netflix, YouTube, Spotify, and other services
PipeWire Audio: Soplos Linux uses PipeWire for optimal audio performance, low latency, and advanced routing capabilities out of the box.

Security & Privacy

Firewall

Soplos Linux includes a pre-configured firewall with both command-line and graphical management:

Graphical Interface (GUFW)

For users who prefer a GUI, Soplos Linux includes GUFW (Graphical Uncomplicated Firewall) - a user-friendly interface for managing firewall rules without terminal commands.

Command Line
# Check firewall status
sudo ufw status

# Enable firewall
sudo ufw enable

Password Security

  • Use strong, unique passwords
  • Enable automatic screen lock
  • Consider using a password manager
