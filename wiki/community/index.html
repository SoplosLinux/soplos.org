Code of Conduct

To ensure a welcoming environment for everyone, we follow a strict Code of Conduct.

Core Values:
  • Be Respectful: Treat everyone with kindness and patience.
  • Be Inclusive: We welcome people of all backgrounds and skill levels.
  • Be Constructive: Critique ideas, not people.

If you witness harassment or unacceptable behavior, please report it to info@soploslinux.com.
